‚ùì Troubleshooting

Can't connect to the system?

  • Make sure you're connected to the Bote WiFi network
  • Try typing http://10.0.0.1:5000 in your browser
  • Clear your browser cache and reload
  • Try a different browser

Bottle not detected?

  • Ensure session is started (click Start Session first)
  • Check if bottle is properly inserted
  • Click "Test Sensor" to verify sensor is working
  • Wait 2-3 seconds between bottle insertions

No internet after claiming bottles?

  • Check if your time display shows remaining time
  • Look for "‚úÖ Internet Active" status
  • Try clicking Refresh button
  • Disconnect and reconnect to WiFi

Time not counting down?

  • The page auto-updates every 0.5 seconds
  • Click Refresh if display seems frozen
  • Check your internet connection
